First setup your virtual environment with python 3.4:

$ pip install virtualenv
$ virtualenv tardigrade
$ cd tardigrade

Then set up your dependencies:

$ Scripts\pip3.4.exe install flask
$ Scripts\pip3.4.exe install Flask-SQLAlchemy
